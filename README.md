# Hostorio - Web Hosting Company Homepage

A modern, responsive homepage for Hostorio.com - a professional web hosting services provider in Bangladesh.

## Features

### Design Elements
- **Modern & Professional**: Clean, uncluttered layout with strong visual hierarchy
- **Brand Colors**:
  - Primary Orange (#FF7800) for CTAs and accents
  - Primary Blue (#0C379A) for main sections
  - White and gradient backgrounds for contrast
- **Fully Responsive**: Mobile-first design that works on all devices
- **Interactive Elements**: Smooth animations, hover effects, and transitions

### Sections

1. **Header**
   - Logo with branding
   - Contact information (phone & email)
   - Professional navigation bar

2. **Hero Section**
   - Eye-catching headline: "Fastest Data Center Web Hosting"
   - Compelling value proposition
   - Strong call-to-action button
   - Dynamic background with animations

3. **Hosting Packages**
   - Four distinct plans (Standard, Professional, Business, Diamond)
   - Clear pricing in BDT (Bangladeshi Taka)
   - Feature comparisons
   - Highlighted "Popular" package
   - Individual "BUY NOW" buttons

4. **Why Choose Us**
   - 8 key selling points with icons
   - Features include: SSL, backups, security, support
   - Dark blue background with white text
   - "GET STARTED NOW" CTA

5. **Softaculous Apps**
   - 380+ one-click install apps
   - Categorized display (CMS, Popular, E-Commerce)
   - Popular apps: WordPress, Joomla, Drupal, Laravel, OpenCart, Magento, PrestaShop

6. **FAQ Section**
   - Accordion-style questions and answers
   - 6 common questions about hosting services
   - Interactive expand/collapse functionality

7. **Footer**
   - Company description
   - Social media links (Facebook, Twitter, YouTube, LinkedIn)
   - Quick links to services
   - Contact information
   - Copyright and legal links

## Customization

### Changing Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-orange: #FF7800;
    --primary-blue: #0C379A;
    --white: #FFFFFF;
    --light-gray: #F8F9FA;
    --text-dark: #2C3E50;
    --text-light: #6C757D;
}
```

### Updating Content
- Edit text directly in `index.html`
- Update pricing in the package cards
- Modify FAQ questions and answers
- Change contact information in header and footer

### Adding Features
- Add more hosting packages by duplicating `.package-card` elements
- Include additional FAQ items
- Expand the Softaculous apps section
- Add testimonials or client logos
